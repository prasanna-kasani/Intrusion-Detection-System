# 🔐 Intrusion Detection System using Machine Learning

## 📌 Project Description
This project is a web-based Intrusion Detection System (IDS) that uses Machine Learning algorithms to classify network activity as normal or malicious. The system helps in detecting potential cyber attacks in real time.

---

## 🚀 Technologies Used
- Python
- Flask (Web Framework)
- Machine Learning (SVM, Ensemble Methods)
- Pandas, NumPy, Scikit-learn
- HTML, CSS

---

## ⚙️ Features
- Detects network intrusions
- Uses SVM with ensemble techniques for better accuracy
- Web interface for user input
- Real-time prediction output
